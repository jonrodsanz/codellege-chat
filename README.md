# Simple chat using Socket.io

![Demo](/src/public/assets/demo-chat.png)

### Install
To install the dependencies, run:
 ~~~
npm install
 ~~~

### Run on the localhost
Run the command:
 ~~~
npm start
 ~~~

### Dependencies:
  - Node:
    - Socket.io
    - Express
    - Body-Parser
    - Mongoose
    - Node-emoji
  - jQuery
  - Pug
  - Bootstrap 4
