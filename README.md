# Dev chat using Socket.io

![Demo](/src/public/assets/Demo.png)

### Install
To install the dependencies, run:
 ~~~
npm install
 ~~~

### Run on the localhost
Run the command:
 ~~~
npm start
 ~~~

### Dependencies:
  - Node (Server-side):
    - Socket.io
    - Express
    - Body-Parser
    - Mongoose
    - Node-emoji
  - (Client-side):
    - jQuery
    - Pug
    - Bootstrap 4
    - Vue
